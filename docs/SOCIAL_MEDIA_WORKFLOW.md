# Social Media Workflow

## Purpose

This workflow turns social media into a managed execution lane inside the Shopify Workflow System.

## Supported Work Categories

- Content strategy
- Post drafting
- Short video prompts
- Image prompts
- Caption writing
- CTA testing
- Comment and reply drafting
- Scheduling records
- Engagement tracking
- Performance reporting
- Repurposing

## Safe Operating Rules

- Create drafts first.
- Do not publish live posts from this bridge.
- Do not send automated replies without approval.
- Do not make unsupported claims.
- Log the source, asset, platform, CTA, and performance result.

## Workflow Mapping

### Content Drafting

- Create platform-specific post batch.
- Create caption batch.
- Create video prompt batch.
- Create image prompt batch.
- Create engagement reply bank.
- Create CTA variants.
- Create hashtag bank.
- Create repurpose map.

### Scheduler Export

- Create scheduling records.
- Assign owner.
- Assign publish window.
- Attach asset URL.
- Track approval status.

### Reporting / Analytics

- Track post URL.
- Track impressions.
- Track clicks.
- Track comments.
- Track saves and shares.
- Track leads.
- Track sales or conversion path.

### Optimization / Repurpose

- Identify winning hooks.
- Improve CTA.
- Repurpose high-performing content.
- Turn audience questions into new content.
- Build the next batch.

## Required Evidence

Each social issue should include platform, campaign, hook, caption, CTA, asset URL, approval status, scheduled date, published URL when available, and performance metric.
